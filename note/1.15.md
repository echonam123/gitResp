## 标题（一级，二级，三级）

## 特殊文本

1. 加粗：左右两边两个*或者两个_或者ctrl+B选中文本

2. 斜体：ctrl+I

3. 斜体并加粗：三个*或者三个下划线包围

4. 删除线：文本左右两边用两个波浪线包围
5. 高亮：高亮用两个=围

## 代码

用“·”包围

代码块：用三个”`"并敲回车

也可以是三个`敲一个c（表示C语言）再回车

```c
int main()
{
    printf("hello world\n");
    return 0;
}
```
